This project is a demonstration of a real world weather app that allows users to search for any city or state in the world, view current weather conditions, check for a five-day weather forecast with respective backgrounds/icons, and manage their favourite location list.
The UI is clean, fast and friendly to any device.

**Futures:**
Real time weather search based on OpenWeather API.
Built in cache for better performance and less API calls.
Responsive UI for any device.
Five-day forecast using scrollable cards.
Favourite locations stored in LocalStorage.
Weather-based dynamic backgrounds, icons, and colors.
Clean reusable components.

**Installation and Running Locally:**

Clone Repo:
git clone https://github.com/ChrisAlymaras/WeatherApp

Navigate into project's file:
cd weather-forecast

Install dependencies:
npm install

OpenWeather API Key:
Use my built in OpenWeather API key or replace with yours in src/api/WeatherApiCall.js & src/api/ComingDaysApiCall.js

Start server:
npm start
